# 家計簿アプリ - Household Account Book

## 概要

**「家計簿アプリ」** は、日々の支出を手軽に記録し、家計管理を効率化するために開発されたモダンなWebアプリケーションです。

これまでExcelや手帳で管理していた手間をなくし、直感的なUIで誰でも簡単に支出管理を始められます。月ごとの支出サマリーやグラフを通じて、自分のお金の流れを正確に把握し、より計画的な家計管理をサポートします。

このプロジェクトは、フロントエンドからバックエンド、データベース設計、ユーザー認証まで、Webアプリケーション開発に必要な技術要素を網羅しており、自身のポートフォリオとして作成しました。

## 主な機能

### 1. ユーザー認証
- **Firebase Authentication** を利用した、安全なメールアドレスとパスワードによるログイン・新規登録機能。
- ユーザーごとにデータが完全に分離され、プライバシーを保護。

### 2. ダッシュボード
- **月次サマリー**: 当月の総支出、予算に対する進捗、一日あたりの平均支出などを表示。
- **支出グラフ**: カテゴリー別の支出割合を示す円グラフや、支払い方法ごとの利用状況がわかるグラフで、支出傾向を視覚的に分析。
- **カレンダー表示**: 支出があった日をカレンダー形式で確認でき、クリックでその日の支出詳細を一覧表示。

### 3. 支出管理
- **支出の記録**: 日付、金額、カテゴリー、支払い方法、店名、メモを記録できる入力フォーム。
- **支出の編集・削除**: 記録した支出は後からいつでも修正・削除が可能。
- **複製機能**: 同じような支出をワンクリックで複製し、入力の手間を削減。

### 4. 定期支出の自動処理
- 家賃や光熱費、サブスクリプションサービスなど、毎月決まった支出を「定期支出」としてテンプレート登録。
- 登録した支払日に基づき、システムが自動で支出を記録。

### 5. 設定
- **カテゴリー管理**: 食費、交通費、趣味など、ライフスタイルに合わせて支出カテゴリーを自由に作成・編集・削除。
- **支払い方法管理**: クレジットカード、電子マネー、現金など、利用する支払い方法を自由に設定。
- **予算設定**: 月ごとの予算を設定し、ダッシュボードで達成状況を管理。

## 使用技術

こだわった点や選定理由などを添えて記載すると、よりアピールに繋がります。

| 領域         | 技術                               |
| :----------- | :--------------------------------- |
| **フロントエンド** | Next.js (App Router), React, TypeScript, Tailwind CSS |
| **バックエンド** | Firebase (Functions - for future extension) |
| **データベース** | Firestore (NoSQL)                  |
| **認証**       | Firebase Authentication            |
| **ホスティング**   | Vercel / Firebase Hosting          |

### 技術選定のポイント
- **Next.js (App Router)**: モダンなReact開発のベストプラクティスを採用し、サーバーコンポーネントとクライアントコンポーネントを適切に使い分けることで、パフォーマンスと開発体験の向上を目指しました。
- **TypeScript**: 静的型付けによる開発時のエラー検知とコードの堅牢性向上を目的として採用しました。
- **Firebase**: バックエンド開発の工数を削減し、フロントエンドの実装に集中するために採用しました。認証からデータベースまで一貫したエコシステムで、迅速な開発を実現しています。
- **Tailwind CSS**: ユーティリティファーストなアプローチにより、CSSの記述量を減らし、効率的で一貫性のあるUIデザインを可能にしました。
