# 収支管理アプリ (Household Account Book)

## 概要

**「収支管理アプリ」** は、日々の家計管理を効率化し、資産形成をサポートするために開発されたモダンなWebアプリケーションです。
直感的な操作性と豊富な分析機能を備え、単なる記録ツールではなく「気づき」を与える家計簿を目指して設計されています。スマートフォン（PWA対応）からPCまで、あらゆるデバイスで快適に利用可能です。

## 主な機能

### 1. ダッシュボード & 資産状況の可視化
- **月次サマリー**: 当月の収入・支出・収支バランスを一目で確認。
- **カード引き落とし管理**: クレジットカードの締め日・支払日を自動計算し、**「いつまでに銀行口座にいくら必要か」**を警告付きで表示。資金不足を未然に防ぎます。
- **グラフ分析**:
  - **収支推移**: 収入と支出のバランスを視覚化。
  - **カテゴリ別支出**: 何にお金を使っているかを円グラフで把握。
  - **店名・サービス別支出**: よく利用する店舗やサービスをランキング形式（棒グラフ）で表示。

### 2. 支出・収入の記録
- **スマート入力フォーム**: 金額入力欄に**電卓機能**（例: `100+250` と入力して計算）を搭載。レシートの合算もアプリ内で完結します。
- **高度な編集機能**: 記録したデータは検索・フィルタリングが可能。複数選択による**一括編集**（カテゴリ変更など）にも対応しています。
- **定期データ管理**: 家賃やサブスクリプションなど、毎月決まった支払いや収入をテンプレート化し、自動入力をサポート。

### 3. カレンダー機能
- **月間カレンダー**: 日ごとの支出額をカレンダー形式で表示。
- **支出履歴の比較**: カレンダー下部に「今月の支出履歴」と**「先月の支出履歴」**を並べて表示し、前月との比較が容易に行えます。

### 4. 年間レポート & 分析
- **年間サマリー**: 年間の総支出、総収入、貯蓄額を自動集計。
- **多角的な分析**:
  - 月別の収支推移グラフ。
  - カテゴリ別、支払い方法別、店舗別の年間支出内訳。
  - 項目が多い店舗別支出も、レスポンシブな棒グラフで見やすく表示。

### 5. カスタマイズ & 設定
- **ダークモード対応**: 夜間の利用でも目に優しいダークテーマを搭載（ヘッダーで切り替え可能）。
- **カテゴリ・支払い方法設定**: 自身のライフスタイルに合わせて自由にカスタマイズ可能。クレジットカード設定では「締め日」「支払日」「支払月（翌月/翌々月）」まで詳細に設定できます。
- **PWA (Progressive Web App) 対応**: スマートフォンのホーム画面に追加することで、ネイティブアプリのように全画面で利用可能です。

## 使用技術

| 領域 | 技術 |
| :--- | :--- |
| **フロントエンド** | Next.js 16 (App Router), React 19, TypeScript |
| **スタイリング** | Tailwind CSS (Dark Mode対応) |
| **グラフ描画** | Recharts |
| **バックエンド (BaaS)** | Firebase (Authentication, Firestore) |
| **日付処理** | date-fns |
| **デプロイ** | Vercel |

### 技術的なこだわり
- **Next.js App Router**: 最新のReact機能（Server Components）を活用し、高速な初期表示とSEOフレンドリーな構成を実現。
- **レスポンシブデザイン**: PCの大画面からスマホの小画面まで、グラフや文字サイズが動的に最適化されるよう設計（Tailwind CSS活用）。
- **PWA対応**: `manifest.json` と `viewport` 設定により、モバイル体験を向上。
- **型安全性**: TypeScriptを全面採用し、堅牢なコードベースを維持。

## ディレクトリ構成

```
src/
├── app/              # Next.js App Router ページコンポーネント
│   ├── dashboard/    # ダッシュボード
│   ├── calendar/     # カレンダー
│   ├── transactions/ # 収支記録
│   ├── yearly-report/# 年間レポート
│   └── ...
├── components/       # 再利用可能なUIコンポーネント
│   ├── dashboard/    # グラフやウィジェット
│   ├── expenses/     # 支出関連フォーム・リスト
│   └── ...
├── lib/              # Firebase設定などのユーティリティ
├── types/            # TypeScript型定義
└── contexts/         # React Context (Auth, Theme)
```

## 今後の展望
- **レシート撮影機能 (OCR)**: カメラでレシートを撮影し、AIが自動で品目と金額を入力する機能の実装。
- **CSVインポート**: 銀行やカード明細のCSV取り込みと自動仕訳機能。
- **資産シミュレーション**: 現在のペースで生活した場合の将来の資産推移予測。
